git 常用命令行操作

发表于 2017-02-22 | 分类于 learn-git | | 阅读次数

git add .

. 表示将整个working directory都提交到staging area

git diff --cached

show what has been staged and what hasn’t been committed

git rm --cached file_name

把文件从stage中删除，但文件本身不删除

# 添加到stage
git add a.txt

# 只从stage中删除， 保留文件
git rm --cached a.txt

git rm -f file_name

强制删除，不保留文件

# 添加到stage
git add a.txt

# 下面从stage中直接删除不行
git rm a.txt

# 使用强制删除，将文件也删除
git rm -f a.txt

git mv file_a file_b

将 文件a 改名为 文件b

git add a.txt

git commit -m "add file a.txt"

git mv a.txt b.txt
